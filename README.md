# Auto-typing-tools

## Requirements:
- Python 3.13.5

## Usage:

#### Set up
```bash
  git clone https://github.com/Kuro-orzz/Auto-typing-tools.git
  cd Auto-typing-tools
  python -m venv venv
  pip install -r requirements.txt
```

#### Run script
```bash
  python main.py --min-delay [min delay] --max-delay [max delay]
```

- After open new window, it wail until you ready.
- If ready, press `F8` key to start auto typing

> ### Note: Do not switch away from the tab, must focus on that tab while tool is working

## New features:

- Add fake typing just in case:))). It's mean that if you press random any key in the keyboard, it will correct it for you. (But just don't spam too fast, need a little delay between keys press)
 
Note: So with this new feature, you don't need to care even if sensei watching you typing =)))
